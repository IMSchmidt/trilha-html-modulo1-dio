Tags de cabeçalho

Voltar

As tags de cabeçalho são usadas para marcar inicio de tópicos ou para títulos, nessa página usamos até esse momento os três maiores cabeçalhos, sendo o <h1> o maior e colocado no topo dessa página, o <h2> usado nos tópicos e o <h3> que estão presentes nas listas. Alem desses podemos usar mais três tamanho o <h4>, <h5>, <h6>, abaixo exemplo de uso desses:

Este texto está usando o <h4>

Este texto está usando o <h5>
Este texto está usando o <h6>

Listas

Voltar

No HTML temos basicamente dois tipos de lista, a ordenada <ol> e a não ordenada <ul>.

Nestá página estamos usando uma lista ordenada no topo como indice e uma lista não ordenada com as referencias. Ambas as listas usam a tag <li> para incluir elementos a lista.

Trabalhando com textos

Voltar

Os textos nessa página estão inseridos dentro da tag de parágrafo <p> isso melhora a estética da página e pode facilitar posteriormente a estilização.

A estilização de texto deixou de ser uma boa prática no HTML, devemos estilizar o texto no CSS, mas apesar de não ser uma boa prática o HTML ainda suporta diversas opções de estilização. As tags mais usadas para estilização de texto são:

  • <i> tag usada para deixar o texto em itálico
  • <u> tag usada para sublinhar o texto
  • <strong> tag usada para deixar o texto com mais "força", basicamente o negrito.
  • <sub> tag usada para escrever o texto um nível abaixo como esse
  • <sup> tag usada para escrever o texto um nível acima como esse
  • <sup> tag usada para marcar um texto, tipo um marca texto.
  • <small> deixa seu texto menor que o da página padrão
  • <blockquote> usado para sitações, eu ainda prefiro as aspas...

Links

Voltar

Links podem ser internos ou externos, o que diferencia é o apontamento. A tag para ambos é a mesma <a>, a diferença é no apontamento dentro do href para link internos a própria página inicia-se com o #, caso o link seja externo necessário colocar o endereço.
